# Roadmap

## MVP

| Feature | Status | Notes |
|---|---|---|
| Registration | ✅ Done | Patient + Professional self-registration on mobile; admin accounts via API |
| Booking | ✅ Done | Create, list, cancel; pessimistic lock prevents double-booking |
| Verification | ✅ Done | Admin verifies professionals via Angular admin panel |
| Chat | ❌ Not started | Real-time messaging between patient and professional |
| Payments | ❌ Not started | Payment gateway integration (CMI / Stripe) |
| Reviews | ❌ Not started | Patients rate professionals after completed bookings |
